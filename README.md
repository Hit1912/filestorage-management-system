# DHR Nest - Premium Cloud Storage Platform

DHR Nest is a state-of-the-art, frontend-only file management and storage system. Designed with a focus on speed, security, and a premium user experience, it allows users to manage their files entirely within the browser using `localStorage` for data persistence.

## 🚀 Key Features

- **🔐 Local Authentication**: Secure login and signup experience managed locally.
- **📊 Advanced Analytics**: Interactive charts and storage usage tracking.
- **📤 Multi-file Management**: Seamlessly upload and manage multiple files at once.
- **📁 Organized Storage**: View files in both Grid and List formats with detailed metadata.
- **🔑 API Key Management**: Generate and manage API keys for external integrations.
- **🔍 Smart Search**: Real-time file searching and pagination.
- **🎨 Premium UI/UX**: Built with React 19, Tailwind CSS v4, and Shadcn/ui for a stunning aesthetic.
- **🌓 Theme Support**: Fully responsive design with Dark and Light mode support.

## 🛠️ Technology Stack

- **Frontend**: React 19, Vite, TypeScript
- **State Management**: Redux Toolkit & RTK Query (Mocked)
- **Styling**: Tailwind CSS v4, Lucide React Icons
- **Components**: Shadcn/ui
- **Persistence**: Browser LocalStorage

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Hit1912/filestorage-management-system.git
   ```

2. **Install dependencies**:
   ```bash
   cd client
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
